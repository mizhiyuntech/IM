# IM

IM 即时通讯 App，基于 React Native + Ant Design 构建。

## 功能

- 登录/退出
- 消息列表（私聊+群聊，未读角标，长按删除/标记已读）
- 聊天页（气泡消息，文字输入发送）
- 通讯录（字母索引，新的朋友/群聊入口）
- 我的（个人信息，设置，关于）

## 技术栈

- React Native 0.85
- @ant-design/react-native 5.4
- React Navigation 7
- Context + useReducer 状态管理

## 开发

```sh
npm install
npm start
npm run android
```
